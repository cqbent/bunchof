== Description ==
<h3>Communicate with your Visitors</h3>
Show every content like Posts, Pages, Costom Post types in lightboxed Dialogs. Like in posts or pages, everything can be shown in a dialog. 
Add welcome screens, videos, forms, polls, just everything WordPress plugins offering you. Open dialogs by visiting a site or by using shortcode links.   

<h3>Layout your dialogs</h3>
By using templates in your theme, style your and change the dialog appearance. 

<h3>Features</h3>

<ul>
	<li>Visual Editor</li>
	<li>Text Editor</li>
	<li>Visual Editor Button to add links for open dialogs</li>
	<li>Open dialogs on site visits</li> 
</ul>

== Installation ==
1. Upload the plugin to the '/wp-content/plugins/' directory.<br>
2. Activate the plugin.

That's it, have fun!
